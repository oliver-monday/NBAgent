| **Picks Review integration — human_verdict field + audit precision + parlay stub + frontend badge (2026-03-16)** | Wires manually-produced daily `data/picks_review_YYYY-MM-DD.json` files into three agents. Review file schema: `[{date, player_name, team, prop_type, pick_value, verdict, trim_reasons}]`; verdicts: "keep"/"trim"/"manual_skip"; join key: `(player_name_lower, prop_type, pick_value)`.
